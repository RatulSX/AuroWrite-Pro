# AuroWrite-Pro
AuroWrite Pro A high-performance mobile first code editor and web IDE. AuraWrite Pro is built for developers who need to build, test, and export production ready web applications directly from a mobile device without sacrificing UI aesthetics or speed.

Technical Features
Execution Engine: Dynamic full-screen code injection for testing complex layouts and scripts without iframe limitations.

Asset Tools: Built-in Image-to-Base64 conversion for embedding assets directly into source files.

Layout Stability: Fixed viewport logic designed to maintain UI element positions when the mobile virtual keyboard is active.

Framework-Free: Developed using vanilla JavaScript to ensure low memory usage and high responsiveness on mobile browsers.

Export System: Support for generating standalone .html files or raw .txt backups.
